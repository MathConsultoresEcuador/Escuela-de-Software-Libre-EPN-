1. Histogramas¶

Un histograma nos permite representar un conjunto de datos numéricos, con barras (juntas), una vez que los hemos agrupado en intervalos o categorías.

Tomamos un conjunto de datos numéricos x, 20 valores numéricos que agruparemos en cinco intervalos de manera arbitraria.

x = c(6.9, 6.2, 5.4, 4.7, 4.8, 6.0, 6.3, 5.0, 4.5, 6.3,

Ejecutamos el comando hist sobre nuestro conjunto x:

In [51]:
hist(x)

Este gráfico consta de barras cuya altura (eje y) es el número de elementos perteneciente a cada uno de los intervalos (eje x). El número de elementos en el eje y se indica como Frecuency y corresponde a la frecuencia absoluta.

En R, los intervalos para los histogramas se calculan usando uno de los siguientes métodos: Sturges, Scott o Freedman-Diaconis que veremos a continuación:

Sturges¶

In [52]:
hist(x, breaks = "Sturges")

Scott¶

In [53]:
hist(x, breaks = "Scott")

Freedman-Diaconis¶

In [54]:
hist(x, breaks = "FD")

Colores¶

Para adicionar colores a los gráficos podemos usar el argumento col indicando, entre comillas y en inglés el nombre del color; es posible ver la lista de colores aplicando el comando colors() en la consola de R o en el siguiente pdf http://www.stat.columbia.edu/~tzheng/files/Rcolor.pdf

In [55]:
hist(x, col = "orange")
In [56]:
hist(x, col = "cornflowerblue")
In [57]:
hist(x, col = "pink")

Etiquetas¶

Es posible usar los argumentos main, xlab e ylab para modificar el título y el nombre de los ejes:

In [58]:
hist(x, col = "beige", main = "Mi histograma",
     xlab = "Intervalos de clase", ylab = "Frecuencia")

Scatterplots¶

Hay muchas maneras de crear un scatterplot en R. la función básica es plot(x, y), donde x y y son vectores numéricos.

plot(x, y, main, xlab, ylab, xlim, ylim, axes)

x es el conjunto de datos cuyos valores son las coordenadas horizontales; y es el conjunto de datos cuyos valores son las coordenadas verticales; main es el título de la gráfica; xlab es la etiqueta en el eje horizontal; ylab es la etiqueta en el eje vertical; xlim es de los límites de los valores de x utilizados para el trazado; ylim es de los límites de los valores de y utilizados para el trazado; axes indica si ambos ejes deben graficarse.

Boxplot¶

Recordemos que el comando summary nos muestra el valor de la media, la mediana, el mínimo, y el máximo de nuestro conjunto de datos. El diagrama de cajas o boxplot permite identificar visualmente todos estos valores y da información con respecto a la distribución de los valores de nuestro conjunto de datos.

Para hacer un diagrama de cajas usamos el comando boxplot:

In [71]:
z = iris[,"Sepal.Width"]
boxplot(z)

  • La línea gruesa es la mediana.

  • La parte inferior de la caja es el primer cuartil (1st Qu.), esto es, el valor que separa al primer 25% de los datos (luego de ordenarlos de menor a mayor).

  • La mediana (corresponde al segundo cuartil) permite dividir al conjunto de datos en dos partes de modo tal que cada parte tendrá el 50% de los datos (luego de ordenar todo el conjunto de datos de menor a mayor).

  • El tercer cuartil (3st Qu.) separa al 75% de los datos una vez ordenados de menor a mayor.

  • La distancia que hay entre el primer y el tercer cuartil se conoce como rango intercuartílico (IQR) y permite tener una idea de la dispersión (acumulación) de los valores alrededor de la mediana, también, permite calcular el tamaño de los “bigotes” (aproximadamente 1.5 veces el IQR).

  • Los valores que escapan a los “bigotes” se consideran valores extremos porque “aparentemente” no forman parte del grupo.

La función par(mfrow = c(2,1)), Divide la ventana de gráficos en dos filas y una columna:

In [72]:
par(mfrow = c(2,1))

La función par(mfrow = c(1,1)), Regresa la ventana a su estado original

In [73]:
par(mfrow = c(1,1))
In [74]:
par(mfrow = c(2,1))
boxplot(z, horizontal = TRUE, ylim = c(2, 4.5))
hist(z)
In [75]:
par(mfrow = c(1,2))
boxplot(z, horizontal = F, ylim = c(2, 4.5))
hist(z)

Para dividir las cajas podemos usar el símbolo “~” que, en R, quiere decir “en función de”.

En nuestro caso será: variable en cuestión (Sepal.Length) ~ variable categórica (Species):

In [76]:
boxplot(iris$Sepal.Length ~ iris$Species)

O en horizontal:

In [77]:
boxplot(iris$Sepal.Length ~ iris$Species,
        horizontal = TRUE)

Para colorear las cajas podemos usar el argumento col o el argumento border para colorear únicamente el borde:

In [78]:
boxplot(iris$Sepal.Length ~ iris$Species, horizontal = TRUE,
        border = c("blue", "green", "red"),
        col    = c("red","cornflowerblue","beige"))

Pie charts¶

Los gráficos circulares o pie charts se crean con la función pie(x, labels =) donde x es un vector numérico no negativo que indica el área de cada corte y labels= anota un vector de caracteres de los nombres para los sectores

In [79]:
slices = c(10, 12,4, 16, 8)
lbls = c("US", "UK", "Australia", "Germany", "France")
pie(slices, labels = lbls, main="Pie Chart of Countries")

Para incluir los porcentajes usamos pct y % y para colores en el gráfico usamos el argumento col:

In [80]:
slices = c(10, 12, 4, 16, 8) 
lbls = c("US", "UK", "Australia", "Germany", "France")
pct = round(slices/sum(slices)*100)
lbls = paste(lbls, pct) 
lbls = paste(lbls,"%",sep="") 
pie(slices,labels = lbls, col=rainbow(length(lbls)),
  	main="Pie Chart of Countries")

Gráfico comparativo de la evolución de las estatutaras de los ecuatorianos.

  1. Definimos el espacio donde se graficarán los datos con la función plot() y la opción type = "n" que implica no graficar los puntos.

  3. Añadimos una leyenda al gráfico con la función legend()

Adicionalmente, lty define el tipo de lineas, lwd define el grosor de la linea y cex define el tamaño de la leyenda:
