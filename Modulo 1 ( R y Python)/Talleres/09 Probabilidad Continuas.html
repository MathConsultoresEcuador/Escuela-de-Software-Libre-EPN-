Distribuciónes continuas¶

A continuación se enlistan las distribuciones continuas que analizaremos en el taller, acompañadas de su nombre en el ambiente de R:

  1. Distribución Uniforme (unif)
  2. Distribución Normal (norm)
  3. Distribución t de Student (t)
  4. Distribución Exponencial (exp)

Para estas distribuciones, R permite permite utilizar las siguientes funciones:

  • Densidad: Permite evaluar una función de densidad $f(y)$ en un punto $y$ dado. En R, antecedemos al nombre de la distribución de probabilidad con la letra "d". Así, utilizaremos la función dnorm si queremos evaluar la función de densidad normal

    $f(y)= \frac{1}{\sigma \sqrt{2\pi}}e^{-\frac{(y-\mu)^2}{2\sigma^2}}$

    en un punto $y$, conociendo los parámetros $\mu$ y $\sigma^2$.

    Advertencia: A diferencia de las distribuciones de probabilidad discretas, la probabilidad en un punto exacto $P(Y=y)=0$ para las distribuciones continuas.

  • Distribución: También conocida como función de distribución acumulada, devuelve probabilidades acumuladas del tipo $P(Y \le y)$. En R, antecedemos al nombre de la distribución de probabilidad con la letra "p".

    Así, si queremos obtener $P(Y \le y)$ para la distribución uniforme, la función a utilizar es punif

  • Cuantiles: Devuelve el valor $y_o$ que se ubica en un cuantil $p$ determinado. En otras palabras, obtenemos el valor $y_o$ que cumple $P(Y \le y_o)= p$. En R, antecedemos al nombre de la distribución de probabilidad con la letra "q".

    Así, si queremos obtener el valor $y_o$ tal que $P(Y \le y_o)= p$, para la distribución exponencial, la función a utilizar es qexp

  • Números aleatorios: Permite simular valores aleatorios de $Y$ que siguen una distribución de probabilidad determinada, conociendo los parámetros de la distribución. En R, antecedemos al nombre de la distribución de probabilidad con la letra "r".

    Así, si queremos obtener valores aleatorios de $Y$, para la distribución t Student, la función a utilizar es rt

1. Distribución Uniforme¶

Sabemos que la variable aleatoria $Y \tilde{} Unif(\theta_1,\theta_2)$ tiene función de densidad:

$$ f(y) = \left\{ \begin{array}{ll} \frac{1}{\theta_2 - \theta_1} &\text{si } \theta_1 \le y \le \theta_2 \\ 0 & \text{en cualquier otro punto} \end{array} \right. $$

Ya que la función de densidad uniforme es una constante que no depende de $y$, la evaluación de la función dará siempre el mismo valor. Utilizamos la función de densidad uniforme de R:

```dunif(x, min = 0, max = 1)```

Donde:

  • x es el valor de $y$ a evaluar.
  • min corresponde al parámetro $\theta_1$
  • max corresponde al parámetro $\theta_2$

Partiendo de una distribución $Unif(\theta_1=5,\theta_2=15)$, se puede evaluar la función de densidad $f(y)$ para una secuencia de puntos $y$ entre 0 y 20. Observamos que el valor obtenido entre el intervalo $[\theta_1=5;\theta_2=15]$ es siempre la constante $\frac{1}{\theta_2 - \theta_1} = \frac{1}{10}=0.1$

In [88]:
y  = seq(5, 15, length=100)
fy = dunif(x=y, min=5, max=15)

Podemos visualizar la función de densidad uniforme graficando los vectores $y$ y $f(y)$ con la la función plot(). El argumento xaxt="n" evita que se grafiquen los valores del eje x.

Dichos valores se añaden manualmente con la función axis(side = 1, at = 0:20, cex.axis=0.8) donde side=1 hace referencia al eje x, at = 0:20 especifica mostrar valores del 0 al 20 y cex.axis=0.8 controla el tamaño de letra.

La función segments() permite graficar líneas entre dos puntos. Los puntos se establecen como coordenadas $(x_0,y_0) \text{ y } (x_1,y_1)$. Utilizamos esta función para graficar las secciones donde la densidad uniforme toma el valor $0$ y para dibujar líneas entrecortadas que permitan visualizar el cambio de comportamiento de la función.

In [89]:
plot(y, fy, type="l",col="red",lwd=2,
     xlab="y",ylab="Densidad",
     xlim = c(0,20), ylim = c(0,0.15),

Supongamos que deseamos conocer $P(7 \le Y \le 11)$ la probabilidad de que la variable alteatoria $Y \tilde{} Unif(\theta_1=5,\theta_2=15)$ tome valores entre $7$ y $11$. Podemos reformular la pregunta y observar que

$P(7 \le Y \le 11) = P(Y \le 11)-P(Y \le 7)$

Por lo que es posible emplear la función de __distribución__ uniforme:

```punif(q, min = 0, max = 1, lower.tail = TRUE)```

Donde:

  • q es el valor particular $y$
  • min corresponde al parámetro $\theta_1$
  • max corresponde al parámetro $\theta_2$
  • lower.tail determina si la probabilidad devuelta es $P(Y \le y)$ (si TRUE) o $P(Y > y)$ (si FALSE )

Para $P(Y \le 11)-P(Y \le 7)$ escribimos:

In [90]:
punif(q=11, min=5, max=15) - punif(q=7, min=5, max=15)
0.4

Para observar $P(7 \le Y \le 11)$ como área bajo la curva de densidad uniforme, utilizamos la función polygon(x, y,border=NA) que dibuja polígonos basado en dos vectores de coordenadas x y y. El argumento border=NA determina el color del borde del polígono. Si su valor es NA, no se dibuja el borde.

# Densidad Uniforme
plot(y, fy, type="l",col="red",lwd=2,

     xlim = c(0,20), ylim = c(0,0.15),

# Líneas Entrecortadas

Observamos que la base del rectángulo es $4$ y la altura $0.1$ por lo que se puede verificar que el área de interés es de $0.4$.

Supongamos que estamos intersados en conocer el valor del quintil 2. En otras palabras, deseamos conocer el valor $y_o$ que cumple con $P(Y \le y_0)= 0.20$.

Para encontrar dicho valor utilizamos la función

```qunif(p, min = 0, max = 1, lower.tail = TRUE)```

Donde:

  • p es la probabilidad a la izquierda/derecha de $y_0$. Ver lower.tail.
  • min corresponde al parámetro $\theta_1$
  • max corresponde al parámetro $\theta_2$
  • lower.tail determina si el valor $y_0$ a devolver, cumple con $P(Y \le y_0)= p$ (si TRUE) o $P(Y > y_0)= p$ (si FALSE )

Para encontrar el valor $y_0$ que cumple con $P(Y \le y_0)= 0.20$, escribimos:

In [92]:
qunif(p=0.20, min = 5, max = 15, lower.tail = TRUE)
7

Comprobamos la respuesta $y_o=7$ con la función de distribución punif(), buscando la probabilidad $P(Y \le 7)$

In [93]:
punif(7, min = 5, max = 15,lower.tail = TRUE)
0.2

Finalmente, podemos generar valores aleatorios de $Y \tilde{} Unif(\theta_1=5,\theta_2=15)$ con la función

```runif(n, min = 0, max = 1)```

Donde:

  • n es el número de datos aleatorios a generar.
  • min corresponde al parámetro $\theta_1$
  • max corresponde al parámetro $\theta_2$

Generamos 1'000.000 de valores aleatorios que siguen la distribución $Unif(\theta_1=5,\theta_2=15)$. Si calculamos la media y la varianza de la simulación, observamos que se aproxima a la esperanza $E(Y)=\frac{\theta_1+\theta_2}{2}=\frac{20}{2}=10$ y varianza $Var(Y)=\frac{(\theta_2-\theta_1)^2}{12}=\frac{100}{12}=8.33$ teóricas.

In [94]:

El argumento prob=TRUE en la función hist() sirve para que el eje vertical devuelva valores en t+erminos de la densidad.

2. Distribución Normal¶

Sabemos que la variable aleatoria $Y \tilde{} Normal(\mu,\sigma^2)$ tiene función de densidad:

$f(y)= \frac{1}{\sigma \sqrt{2\pi}}e^{-\frac{(y-\mu)^2}{2\sigma^2}}$ para $-\infty

Supongamos que $Y \tilde{} Normal(\mu=68,\sigma^2=4^2)$ y que estamos interesados en evaluar la función de densidad normal en un punto, digamos $y=75$. Utilizaremos la función de densidad normal

```dnorm(x, mean = 0, sd = 1)```

Donde:

  • x es el valor de $y$ a evaluar.
  • mean es la media $\mu$
  • sd es la desviación estandar $\sigma$

Escribimos:

In [96]:
media = 65
ds    = 4
dnorm(x = 75, mean = media, sd = ds)
0.00438207512339214

También se puede evaluar una vector de puntos. Partimos de una secuencia de números negativos y positivos en el intervalo $[-4;4]$ simulando parte del dominio de una distribución normal estándar $Z \tilde{} Normal(\mu=0,\sigma^2=1)$. Para que el dominio se adapte a la distribución $Y \tilde{} Normal(\mu=68,\sigma^2=4^2)$ es necesario multiplicar a cada uno de los valores de la secuencia por $\sigma$ y sumarles $\mu$.

En otras palabras, realizamos la transformación inversa desde una distribución normal estándar a una distribución normal cualquiera:

Si $Z = \frac{Y-\mu}{\sigma}$ entonces $Y = Z\sigma + \mu$

Podemos visualizar la función de densidad de $Y \tilde{} Normal(\mu=68,\sigma^2=4^2)$ graficando los vectores $y$ y $f(y)$ con la la función plot(). Observamos que la distribución se encuentra centrada en $\mu=65$.

Supongamos que deseamos conocer $P(Y > 70)$ la probabilidad de que la variable alteatoria $Y$ tome valores mayores a $70$. Podemos emplear la función de distribución normal:

```pnorm(q, mean = 0, sd = 1, lower.tail = TRUE)```

Donde:

  • q es el valor particular $y$
  • mean es la media $\mu$
  • sd es la desviación estandar $\sigma$
  • lower.tail determina si la probabilidad devuelta es $P(Y \le y)$ (si TRUE) o $P(Y > y)$ (si FALSE )

Para $P(Y > 70)$ escribimos:

In [99]:
pnorm(q=70, mean = media, sd = ds, lower.tail=F)
0.105649773666855

Para observar $P(Y > 70)=0.1056$ como área bajo la curva de densidad normal, utilizamos la función polygon(x, y) que dibuja polígonos basado en dos vectores de coordenadas x y y.

En este caso, las coordenadas de $x$ van desde $70$ en adelante. Para $y$, las coordenadas van desde 0 e incluyen todos los puntos evaluados $f(y)$ donde $y>70$.

In [100]:
plot(y, fy, type="l",col="red",lwd=2,

y.cord = c(0 , fy[y>70])
polygon(x.cord, y.cord, border=NA, col="red")

Supongamos que estamos intersados en conocer el valor del decil 4. En otras palabras, deseamos conocer el valor $y_o$ que cumple con $P(Y \le y_0)= 0.40$.

Para encontrar dicho valor utilizamos la función

```qnorm(p, mean = 0, sd = 1, lower.tail = TRUE)```

Donde:

  • p es la probabilidad a la izquierda/derecha de $y_0$. Ver lower.tail.
  • mean es la media $\mu$
  • sd es la desviación estandar $\sigma$
  • lower.tail determina si el valor $y_0$ a devolver, cumple con $P(Y \le y_0)= p$ (si TRUE) o $P(Y > y_0)= p$ (si FALSE )

Otra manera de encontrar el decil 4 sería buscar el valor $y_0$ que cumple con $P(Y > y_0)= 1-0.40=0.6$, escribimos:

In [101]:
qnorm(p=0.6,  mean = media, sd = ds, lower.tail = F)
63.9866115874568

Comprobamos la respuesta $y_o\approx63.99$ con la función de distribución pnorm(), buscando la probabilidad $P(Y \le 63.99)$

In [102]:
pnorm(63.99, mean = media, sd = ds,lower.tail = TRUE)
0.400327307053087

Finalmente, podemos generar valores aleatorios de $Y \tilde{} Normal(\mu=68,\sigma^2=4^2)$ con la función

```rnorm(n, mean = 0, sd = 1)```

Donde:

  • n es el número de datos aleatorios a generar.
  • mean es la media $\mu$
  • sd es la desviación estandar $\sigma$

Generamos 1'000.000 de valores aleatorios que siguen la distribución $Y \tilde{} Normal(\mu=68,\sigma^2=4^2)$, gráficamos un histograma de los valores simulados y sobreponemos la curva de densidad normal. Observamos que la curva de densidad normal se ajusta de buena manera a los datos del histograma.

3. Distribución t Student¶

Sabemos que la variable aleatoria $Y \tilde{} t(v)$ tiene distribución t de Student con $v$ grados de libertad. No trabajaremos evaluando la función de densidad de esta variable para un punto $y$ sino solamente con los comandos de R para obtener gráficos.

Supongamos que $Y \tilde{} t(v=8)$ y que estamos interesados graficar su función de densidad. Al igual que en las distribuciones anteriores, es necesario crear una sequencia y evaluar la función de densidad t para posteriormente realizar un plot. Utilizamos entonces la función:

```dt(x,df)```

Donde:

  • x es el valor de $y$ a evaluar.
  • df son los grados de libertad de la distribución $v$

Para $Y \tilde{} t(v=8)$ escribimos:

In [104]:
y  = seq(-4, 4, length=100)
fy_t8 = dt(x=y,  df=8)

Utilizamos la función plot() para graficar los puntos $y$ y $f(y)$. Observamos que la distribución se encuentra centrada en cero al igual que la distribución normal estándar.

In [105]:
plot(y, fy_t8, type="l",col="red",lwd=2,
     xlab="y",ylab="Densidad",
     main= "Función de densidad t, v=8")

La distribución $t$ también es simétrica y tiene forma de campana, sin embargo las colas de la distribución son más largas que la distribución normal. A medida que los grados de libertad $v=n-1$ aumentan (el número de datos es grande), la distribución $t$ converge con la distribución normal.

Gráficamente, compararemos la distribución normal estándar con la distribución $t$ con distintos grados de libertad. Observamos que con 30 grados de libertad, la distribución $t$ se asemeja bastante a la distribución normal estándar.

In [106]:
# Secuencia
y  = seq(-4, 4, length=100)

# Distribuciones t con diferentes grados de libertad
fy_t1  = dt(x=y,  df=1)
fy_t3  = dt(x=y,  df=3)
fy_t8  = dt(x=y,  df=8)
fy_t30 = dt(x=y,  df=30)

# Distribución normal estandar
fy_n   = dnorm(x=y, mean=0, sd=1 )

# Plot distribución normal estandar
plot(y, fy_n, type="l", lty=2, xlab="y",
  ylab="Densidad", main="Comparación de distribuciones t")

# Incluir distribuciones t
lines(y, fy_t1, lwd=2, col="red")
lines(y, fy_t3, lwd=2, col="blue")
lines(y, fy_t8, lwd=2, col="darkgreen")
lines(y, fy_t30,lwd=2, col="gold")

# Leyenda del gráfico
etiquetas = c("df=1", "df=3", "df=8", "df=30", "normal")

legend("topright", inset=0.05, title="Distribuciones",
      legend=etiquetas, lwd=2, lty=c(1, 1, 1, 1, 2),
      col= c("red", "blue", "darkgreen", "gold", "black"))

Las funciones de dsitribución, cuantiles y generación de números aleatorios están también disponibles para esta distribución sin embargo, no hacen parte de este taller. Para más información consultar la ayuda de las funciones pt, qt y rt.

4. Distribución exponencial¶

Sabemos que la variable aleatoria $Y \tilde{} exp(\beta)$ tiene función de densidad:

$$ f(y) = \left\{ \begin{array}{ll} \frac{1}{\beta} e^{-y/\beta}&\text{si } 0 \le y < \infty \\ 0 & \text{en cualquier otro punto} \end{array} \right. $$

Para graficar la función de densidad utilizamos nuevamente la función de densidad de R:

```dexp(x, rate = 1)```

Donde:

  • x es el valor de $y$ a evaluar.
  • rate es constante $1/\beta$

Para $Y \tilde{} exp(\beta=3)$ escribimos:

In [107]:
y  = seq(0, 20, length=1000)
fy = dexp(x=y,  rate = 1/3)

Graficamos la función de densidad en dos estapas. En la primera utilizamos la función de densidad obtenida con dexp() misma que evalúa la función de densidad para valores superiores a 0. En la segunda etapa utilizamos nuevamente la función segments() para añadir una línea recta que represnta el valor de la función exponencial para los valores inferiores a 0.

También añadimos una línea entrecortada que ayuda avisualizar el cambio de comportamiento de la función. Recordemos que la función de densidad exponencial interseca al eje vertical en el punto $1/\beta$.

Para $Y \tilde{} exp(\beta=3)$ escribimos:

In [108]:
# Para valores mayores a 0
plot(y, fy, type="l",col="red",lwd=2,
     xlim = c(-5,20), ylim = c(0,0.35),
     xlab="y",ylab="Densidad",
     main= expression(paste("Función de densidad exponencial, ", beta, "=3 ")))

# Para todos los valores inferiores a 0
segments(x0=-5 ,y0=0,x1=0 ,y1=0,col ="red",lwd = 2)

# Línea entrecortada
segments(x0=0 ,y0=0, x1=0, y1=1/3,col ="red",lty=2)

Supongamos que deseamos conocer $P(Y > 10)$ la probabilidad de que la variable alteatoria $Y$ tome valores mayores a $10$. Podemos emplear la función de distribución exponencial:

```pexp(q, rate = 1 lower.tail = TRUE)```

Donde:

  • q es el valor particular $y$
  • rate es constante $1/\beta$
  • lower.tail determina si la probabilidad devuelta es $P(Y \le y)$ (si TRUE) o $P(Y > y)$ (si FALSE )

Para $P(Y > 10)$ escribimos:

In [109]:
pexp(q=10, rate = 1/3, lower.tail=F)
0.0356739933472524

Para observar $P(Y > 10)=0.0357$ como área bajo la curva de densidad exponencial, utilizamos la función polygon(x, y) que dibuja polígonos basado en dos vectores de coordenadas x y y.

En este caso, las coordenadas de $x$ van desde $10$ en adelante. Para $y$, las coordenadas van desde 0 e incluyen todos los puntos evaluados $f(y)$ donde $y>10$.

In [110]:
# Para valores mayores a 0

Supongamos que estamos intersados en conocer el valor del cuartil 3. En otras palabras, deseamos conocer el valor $y_o$ que cumple con $P(Y \le y_0)= 0.75$.

Para encontrar dicho valor utilizamos la función

```qexp(p, rate = 1, lower.tail = TRUE)```

Donde:

  • p es la probabilidad a la izquierda/derecha de $y_0$. Ver lower.tail.
  • rate es constante $1/\beta$
  • lower.tail determina si el valor $y_0$ a devolver, cumple con $P(Y \le y_0)= p$ (si TRUE) o $P(Y > y_0)= p$ (si FALSE )

Para enocntrar el valor $y_0$ que cumple con $P(Y \le y_0)= 0.75$, escribimos:

In [111]:
qexp(p=0.75, rate= 1/3, lower.tail = T)
4.15888308335967

Comprobamos la respuesta $y_o\approx 4.16$ con la función de distribución pexp(), buscando la probabilidad $P(Y \le 4.16)$

In [112]:
pexp(4.16, rate= 1/3, lower.tail = TRUE)
0.750093059062417

Finalmente, podemos generar valores aleatorios de $Y \tilde{} exp(\beta = 3)$ con la función

```rexp(n, rate = 1)```

Donde:

  • n es el número de datos aleatorios a generar.
  • rate es constante $1/\beta$

Generamos 1'000.000 de valores aleatorios que siguen la distribución $Y \tilde{} exp(\beta = 3)$, gráficamos un histograma de los valores simulados y sobreponemos la curva de densidad exponencial.

Observamos que la curva de densidad normal se ajusta de buena manera a los datos del histograma.
